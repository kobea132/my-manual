###github 初始化設定

1.  先建立 github repostory
2.  clone 至本地端, clone 下來後會在當前目錄下建立一個 **_my-manual_** 目錄
    ```sh
    git clone https://github.com/kobea132/my-manual.git
    ```
3.  新增檔案 _index.html_ 至 git
    ```sh
    git add index.html
    ```
4.  提交版本
    ```sh
    git commit -m "init"
    ```
5.  上傳至github 主機
    ```sh
    git push origin master  
    ```
